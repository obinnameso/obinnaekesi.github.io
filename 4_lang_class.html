Language Identification
Challenge

Project Description

Welcome to our project, Language as you know is an indispensable tool that can be used to deepen democracy and also contribute to the social, cultural, intellectual, economic and political life of the South African society. This project aims to develop a language identification system that can accurately identify South African languages of a given text input. The system utilizes natural language processing (NLP) techniques to analyze text features and determine the most likely language.

South Africa is multilingual with 11 official languages, each of which is guaranteed equal status. Most South Africans are multilingual and able to speak at least two or more of the official languages. With such a multilingual population, it is only obvious that our systems and devices also communicate in multi-languages. In this project, we will take text which is in any of South Africa's 11 Official languages and identify which language the text is in.

Features

Identifies the language of a given text input Supports any of South Africa's 11 Official languages Employs NLP techniques to analyze text features and determine language probabilities Provides a user-friendly interface for language identification.

Data

The system requires a dataset of text samples labeled with their corresponding South African languages. The dataset covers diverse range of text genres and represent the linguistic diversity of South Africa.
The data used here contained equal samples of each language to enhance proper and balanced training of the model as seen in the graph below:

Model

Various models were tested but the Naive Bayes classifier performed better so the system employs the Naive Bayes classifier, which was trained on a labeled dataset of South African text samples. The model utilizes NLP techniques, such as n-gram analysis and character frequency distribution, to extract features from the text and make language predictions.

Evaluation

The system's performance was evaluated using metrics such as accuracy, precision, recall, and F1-score. These metrics assessed the system's ability to correctly identify the language of text samples.
